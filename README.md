# FIAP Bank — Tech Challenge (Frontend)

Implementação do Tech Challenge: Next.js + TypeScript + Tailwind + Storybook.

## Recursos
- Boas-vindas (abre logado) com animação (framer-motion)
- Home: saldo + últimas transações + modal para adicionar transação
- Transações: listagem, editar (modal), excluir
- Design System simples em `src/components` (Button, Card, Modal, etc.)
- Mock de dados: `public/transactions.json`
- Storybook configurado (componentes -> stories)
